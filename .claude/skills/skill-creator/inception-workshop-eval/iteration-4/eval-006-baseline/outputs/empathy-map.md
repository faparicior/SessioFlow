# HomeHub - Empathy Map

## Elderly User Concerns

### What They Say
- "This is too complicated"
- "I need help with this"

### What They Think
- "Technology is hard"
- "I should ask my kids"

### What They Do
- Ask for help
- Avoid using features

### What They Feel
- Frustrated
- Confused

## General Observations
- Need simpler interfaces
- Prefer physical buttons
- Need more help
