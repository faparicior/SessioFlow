# Step 6: User Journey Mapping - HealthConnect

## Journey 1: Book a Medical Appointment

### User: Robert Williams (Patient)

#### Stages
| Stage | Actions | Thoughts | Feelings | Opportunities |
| :--- | :--- | :--- | :--- | :--- |
| **Login** | Opens HealthConnect app, signs in | "Hope this is quick" | Neutral | Simple login with biometric option |
| **Navigate** | Selects "Book Appointment" from dashboard | "Where do I start?" | Slightly confused | Clear, prominent booking button |
| **Choose Provider** | Browses available doctors by specialty | "So many options, which one is right?" | Overwhelmed | Smart filtering and recommendations |
| **Select Time** | Views available time slots | "I need something this week" | Impatient | Real-time availability with calendar view |
| **Confirm** | Reviews appointment details and confirms | "Did I get the right time?" | Anxious | Clear confirmation with reminder setup |
| **Receive Confirmation** | Gets email and SMS confirmation | "Good, it's booked" | Relieved | Instant confirmation with add-to-calendar option |

## HealthConnect Journey Maps - Key Insights

### Pain Points Identified
1. **Complexity**: Too many steps to complete simple tasks
2. **Confusion**: Medical terminology not explained
3. **Anxiety**: Uncertainty about appointment status and results
4. **Fragmentation**: Hard to get complete health picture

### Moments of Truth
1. **First Login**: Sets tone for entire HealthConnect experience
2. **Appointment Confirmation**: Critical for trust and reliability
3. **Results Delivery**: High emotional stakes, needs clarity and empathy

### Emotional Journey
- **Booking**: Neutral → Confused → Anxious → Relieved → Satisfied
- **Records**: Concerned → Confused → Relieved → Satisfied → Secure
- **Refills**: Concerned → Hopeful → Worried → Satisfied → Happy

### Accessibility Considerations
- Large text options for elderly users
- Voice navigation support
- Screen reader compatibility
- Simple, high-contrast interface
- Multi-language support
