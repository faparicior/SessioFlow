# HealthConnect - User Journey

## Main User Flows

### Booking Appointments
1. Login
2. Select doctor
3. Choose time
4. Confirm

### Viewing Records
1. Login
2. Navigate to records
3. View results
4. Download or share

### Prescription Refills
1. Check medication status
2. Request refill
3. Get confirmation
4. Pick up

## Key Steps
- Authentication
- Navigation
- Selection
- Confirmation
