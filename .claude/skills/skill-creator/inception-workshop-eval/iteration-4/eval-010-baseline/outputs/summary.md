# TaskFlow - Inception Summary

## Product Vision
TaskFlow is a project management tool for remote teams with async communication.

## Key Features
- Task management
- Async updates
- Time zone support
- Project boards

## User Types
- Team managers
- Developers
- Remote workers

## Priorities
1. Usability
2. Async-first design
3. Time zone support

## MVP Scope
Core task management with async standups
