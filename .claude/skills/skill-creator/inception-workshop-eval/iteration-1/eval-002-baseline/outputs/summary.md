# SessioFlow - Inception Workshop Summary

## Product Vision
SessioFlow is a Call-for-Papers platform for event organizers to manage submissions, reviews, and scheduling.

## Key Features
- Submission management
- Reviewer assignment
- Schedule generation
- Email notifications

## User Types
- Event organizers
- Reviewers
- Submitters

## Priorities
1. Usability
2. Time savings
3. Reliability

## MVP Scope
Core submission and review workflow with basic scheduling.
