## Journey 1: Submit a Proposal

### User: Maria Garcia (Submitter)

| Stage | Actions | Thoughts | Feelings | Opportunities |
| :--- | :--- | :--- | :--- | :--- |

## Journey Maps - Key Insights

### Pain Points Identified
1. **Submission**: Uncertainty about correctness, lack of status visibility
2. **Review**: Unclear time commitment, fragmented communication
3. **Organizer**: Manual processes, time pressure, error risk

### Moments of Truth
1. **Submission confirmation** - First impression of system quality
2. **Review assignment** - Critical for fairness and efficiency
3. **Schedule generation** - Highest value automation opportunity

### Emotional Journey
- **Submitter**: Curious → Anxious → Relieved → Impatient → Satisfied
- **Reviewer**: Neutral → Willing → Engaged → Accomplished → Satisfied
- **Organizer**: Focused → Hopeful → Concerned → Careful → Stressed → Relieved
